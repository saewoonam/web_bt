NIST Encounter tracing programs for testing

This website is hosted on github and contains links to html code to communicate with NIST bluetooth devices being used for exposure/encounter tracing

The link below that communicate only works in Google Chrome Browsers on computers and Android devices. We are working on a app for iOS and Androids for more complete control on those platforms
